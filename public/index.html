양식 자동 감지 — 머리글로 셀러페이/네이버를 구분합니다. 네이버 암호화 파일(.xlsx, OLE2)은 입력한 암호로 브라우저에서 복호화합니다(1행 안내문 자동 건너뜀, 2행을 머리글로 인식).

거래처코드는 셀러페이 7808703064, 네이버 12685로 채널에 맞춰 자동 적용. 출하창고 00003 · 결재방식 온라인 공통.

품목코드 — 네이버는 옵션관리코드를 그대로 사용(품목현황 존재 시 확정). 셀러페이는 상품명·옵션을 품목현황과 대조(색상·형태/세트 사이즈)해 결정. 복합옵션·미확정은 확인필요로 표시되며 클릭해 직접 변경, 수정값은 브라우저에 저장됩니다.

발주서 — 주문번호 1건 = 1행, 옵션 합산 물품명, 수량 합계.

이카운트 — 옵션 1줄 = 1행. 일자 ← 주문/결제일·직접지정 · 적요 ← 네이버는 상품주문번호·셀러페이는 주문번호 · 규격 ← 빈칸(미입력). 금액: 셀러페이는 총결제금액(라인은 상품금액 비율로 배분), 네이버는 최종 상품별 총 주문금액 기준. 부가세 포함가면 공급가액=금액÷1.1, 부가세=차액, 단가=공급가액÷수량.
